Git

opkg update
opkg install git

git clone https://github.com/usuario/repositorio.git /ruta/destino



Asegúrate de que cron esté habilitado en la Onion Omega

/etc/init.d/cron enable
/etc/init.d/cron start

crontab -e

En el archivo crontab, agrega la siguiente línea:


# Actualiza el repositorio cada 6 horas
0 */6 * * * cd /ruta/al/repositorio && git pull

# Ejecuta el script de verificación después de la actualización
5 */6 * * * /ruta/al/script/check_meter_reading.sh

Este comando se ejecutará a la hora 0 de cada 6 horas 
(es decir, a las 0:00, 6:00, 12:00, y 18:00) todos los días.

0 */6 * * *: especifica que el comando se ejecute cada 6 horas.
/ruta/al/script/check_meter_reading.sh: reemplaza esta ruta 
con la ruta completa al script .sh que configuraste.


Después de añadir la línea, guarda y cierra el archivo (Ctrl + O y Ctrl + X en nano).


Verifica que la tarea esté programada
Para asegurarte de que el cron job esté activo,
 puedes listar las tareas cron configuradas con:

crontab -l
